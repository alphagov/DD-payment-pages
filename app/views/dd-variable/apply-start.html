Use Direct Debit to pay the Dartford Crossing charge (Dart Charge)

You can pay the Dartford Crossing charge by variable Direct Debit.

This means your monthly payment can go up or down depending on how many times you are charged.

Charge per crossing:
£2.50

Payments are taken by Direct Debit on the last working day of every month.

Start now

Before you start

To set up a variable Direct Debit payment you’ll need to have the following details ready:

  • Bank account number and sort code
  • Name on account
  • Vehicle registration

If more than one person is required to authorise Direct Debits on the bank account, you must use a different payment method.
